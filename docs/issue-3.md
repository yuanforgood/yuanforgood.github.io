# 信息资讯

- 搜索效率：**谷歌**>**微信搜一搜**
- 搜索方法：

1. **限定关键词**：对关键词加上`“”`双引号
2. **限定标题**：`intitle:`
3. 包含多个关键词：`alintitle`
4. 限定**文章内容**：`intext`
5. 限定**网址**：`inurl`
6. 限定具体完整的网址：`site:`
7. 限定搜索的**图片大小**：`imagesize`
8. 限定**文档**的格式：`filetype:pdf`
